# navodaya-scholar-api

A lightweight RESTful CRUD API built entirely from scratch to manage scholar records.

## 🚀 Tech Stack
* **Language:** Python
* **Framework:** Flask
* **Database:** SQLite3
* **Testing Tool:** Thunder Client

## 🧠 What I Learned / Solved in this Project
* Implemented full **CRUD matrix functionality** (`GET`, `POST`, `PUT`, `DELETE`).
* Enforced structural data integrity using **Parameterized SQL queries** to prevent SQL injection.
* Managed route parameter type enforcement using Flask path variables (e.g., `<int:sno>`).
* Debugged client-server communication issues by verifying Content-Type headers and JSON syntax structures.
